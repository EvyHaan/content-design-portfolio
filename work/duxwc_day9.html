Daily UX Writing Challenge: Day 9

Error message - form field

(concept piece)

The Problem

The Scenario

The user is trying to rent a car using an application but the credit card on file has expired.

The Challenge

Write them an error message so that they can correct the problem.

Headline: 30 characters max
Body: 45 characters max

What I Did

  • UX copy design

The Solution

Microcopy under the relevant option provides the user context.

I want to be succinct, but also want to provide the user with actionable steps to remedy the situation. I offer two options so they can make the choice that is right for them.

The Big 'So What?'

There might be multiple nuanced solutions to a problem. We may not know what solution best applies to a user, but we can get them started on the path that is right for them.
